# Welcome to our EventSync project

Here’s a **description** for **EventSync** that incorporates all the details you provided, including its purpose, features, development tools, and deployment:

---

**EventSync** is a **progressive web app (PWA)** designed to simplify event coordination and shared experiences for friends and families. Built with a focus on ease of use and seamless functionality, EventSync empowers users to:
- **Create, manage, and join events** effortlessly, with tools for scheduling, invitations, RSVPs, and reminders.
- **Share and relive memories** through collaborative photo-sharing, where users can upload, organize, and view photos in event-specific galleries.

### **Key Features**
- **Event Management:**
  - Create events with details like date, location, and agenda.
  - Invite guests via email or shareable links.
  - Track RSVPs and send reminders.
- **Photo Sharing:**
  - Upload and organize photos in event galleries.
  - Privacy controls for public or private events.
- **Cross-Device Accessibility:**
  - Works seamlessly on mobile, tablet, and desktop.
  - Offline access to event details and photos.

### **Development Details**
EventSync was fully developed using **Antigravity**, an advanced agentic AI coding assistant that streamlines the creation of modern, user-friendly applications.

**Design & Architecture:**
- **UI/UX Design:** Strictly follows **Google's Material Design Standards** to ensure a familiar, intuitive, and polished user experience.
- **Supabase Backend:** The app leverages **Supabase** as its comprehensive backend solution, providing:
    - **Database:** A robust PostgreSQL database for real-time event and profile management.
    - **Authentication:** Secure user handling and row-level security.
    - **Storage:** Scalable media storage for event photos and avatars.

### **Deployment**
EventSync is deployed on **Netlify**, a leading platform for modern web applications. Netlify’s seamless integration with PWAs ensures fast loading times, reliable performance, and easy updates, making EventSync accessible to users worldwide.

### **Why EventSync?**
EventSync is more than just an event management tool—it’s a platform designed to bring people together. Whether it’s a birthday party, family reunion, or casual get-together, EventSync ensures that planning and sharing memories are as enjoyable as the events themselves.
